- Cek permission Windows Camera settings

Beberapa antivirus mungkin flag executable PyInstaller. Ini false positive umum. Anda bisa:

- Cek VirusTotal untuk verifikasi
- Build dari source (lihat repo)

- Turunkan brightness/quality webcam
- Pastikan lighting cukup

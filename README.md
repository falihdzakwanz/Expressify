# Expressify
Expressify adalah sebuah permainan dimana kamu dapat menunjukkan seberapa cepat kamu bisa berubah wajah! Dalam 20 detik, kamu akan diberi instruksi acak seperti senyum lebar, cemberut sedih, atau kaget maksimal! setiap ekspresi benar menambah skor, lengkap dengan musik dan efek unik tiap mood.
